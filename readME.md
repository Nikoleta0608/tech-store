# Tech Store – Docker Project

## Описание
Tech Store е уеб приложение за онлайн магазин за техника.  
Проектът е реализиран с PHP и MySQL и е контейнеризиран с Docker и Docker Compose.

## Използвани технологии
- PHP и Apache
- MySQL 
- Docker
- Docker Compose

## Архитектура
Проектът съдържа два основни компонента:

### Backend (PHP + Apache)
- показва уеб страницата
- извлича данни от база данни
- визуализира продукти със снимки

### Database (MySQL)
- съхранява продуктите
- съдържа таблица `products`
- връзка към backend чрез Docker мрежа

## Комуникация между услугите
Backend се свързва с базата данни чрез:
- host: db
- database: tech_db
- user: user
- password: password

Контейнерите комуникират чрез вътрешната Docker мрежа.

## Стартиране на проекта
- docker compose up --build
- http://localhost:8080

## Функционалности
- Показване на продукти  
- Свързване към база данни  
- Автоматично създаване на таблица  
- Динамично извеждане на данни от MySQL  
- Показване на изображения  
- Модерен дизайн (карти)

## Docker
Проектът използва:
- PHP контейнер (Apache сървър)
- MySQL контейнер (база данни)
Контейнерите се стартират и управляват чрез Docker Compose.

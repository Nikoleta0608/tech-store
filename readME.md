# Tech Store – Docker Project

## Описание
Tech Store е уеб приложение, което представя онлайн магазин за техника (телефони и смарт часовници).  
Проектът е реализиран с PHP и MySQL и е контейнеризиран чрез Docker и Docker Compose.

## Технологии
- PHP и Apache 
- MySQL 
- Docker
- Docker Compose


## Архитектура
Проектът съдържа два компонента:

### Backend (PHP + Apache)
- изпълнява PHP кода
- показва съдържанието в браузъра
- прави връзка с базата данни

### Database (MySQL)
- съхранява данни
- стартира като отделен контейнер

## Комуникация

Backend се свързва към базата чрез:
host: db
Контейнерите комуникират чрез вътрешната мрежа на Docker.


## Как работи
PHP файлът:
- създава връзка към MySQL
- проверява дали има връзка
- показва списък с продукти

MySQL:
- създава база данни с име `tech_db`
- приема връзки от backend
